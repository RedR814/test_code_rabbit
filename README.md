# 红黑树实现

本项目包含一个完整的红黑树实现，具有以下特性：

- 插入操作
- 删除操作
- 查找操作
- 获取最小值
- 中序遍历

## 文件说明

- `RedBlackTree.java`: 红黑树的完整实现
- `RedBlackTreeTest.java`: 测试代码，演示了红黑树的基本操作

## 运行测试

```bash
javac -encoding UTF-8 RedBlackTree.java RedBlackTreeTest.java
java RedBlackTreeTest
```

测试程序将演示插入、查找、删除和遍历等操作。
